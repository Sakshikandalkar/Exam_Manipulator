# Exam Manipulator

## 📌 Project Description
Exam Manipulator is a web-based application developed using JavaScript, HTML, and CSS. It is designed to manage and manipulate exam-related functionalities efficiently.

## 🚀 Features
- User-friendly interface
- Interactive functionality using JavaScript
- Responsive design
- Easy to use and manage

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Structure
- index.html
- style.css
- script.js
- other project files

## ▶️ How to Run
1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in your browser
